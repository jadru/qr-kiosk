# QR 코드 기반 키오스크

이 프로젝트는 사장님이 메뉴판과 가개 정보를 직접 수정하여 메뉴판을 수정하고 QR코드를 생성하고 프린트할 수 있고, 사용자들은 토스페이먼츠 모듈을 통해 메뉴를 주문하고 주문 내역을 사장님이 처리하게 만드는 프로젝트입니다.

## Tech Stack
### Backend
- Nest.js
- Typescript
- PostgreSQL
- TYPEORM
- Prisma
### Frontend
- React.js
- Vite
- Typescript
- React Beutiful DND
- Recoil
- Axios
### Deployment
- Yarn Berry (Mono-repo)
- Docker
